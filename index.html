Duplos

Colophon

Duplos - Ricardo Mestre’s Miscellany - was born in 2006, and this design is version 4.2.

It uses HTML5 for markup and CSS3 for presentation. Except for analytics and share buttons, it's bitmap and JavaScript free.

Duplos' previous iteration was showcased on several web design galleries/lists and has won the Awwwards' Site of the Month February 2010. It was also featured on Aarron Walter's book Designing for Emotion.
